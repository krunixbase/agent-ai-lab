# Backup & Migration Archive

This directory contains historical versions of architecture documents and files that were reorganized or deprecated during repository migrations.

---

## Purpose

- Preserve older versions of documents  
- Maintain traceability of architectural evolution  
- Provide reference material for future audits or research  
- Ensure no information is lost during restructuring  

---

## Contents

- Legacy documents  
- Deprecated drafts  
- Files moved during `move.log` and `move2.log` migrations  
- Early prototypes of architectural models  

---

## Notes

- Files in this directory **should not be edited**.  
- New deprecated files should be moved here instead of being deleted.  
- Migration logs in the repository root describe when and why files were moved.
