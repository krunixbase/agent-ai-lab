# Support Guide  
Agent AI Lab — Support and Help  
Version 1.0.0

This document explains how to get help, report issues, and request guidance when working with the Agent AI Lab project.

---

## 1. Where to Ask Questions

### GitHub Discussions
Use Discussions for:

- general questions  
- architecture clarification  
- design proposals  
- best practices  
- feedback  

### GitHub Issues
Use Issues for:

- bugs  
- documentation errors  
- missing links  
- structural inconsistencies  
- safety or governance concerns  

---

## 2. Reporting Problems

Before opening an issue:

- check existing issues  
- check the documentation index  
- verify the problem is reproducible  

When reporting:

- include clear steps  
- include file paths  
- include expected vs actual behavior  
- attach logs or screenshots if relevant  

---

## 3. Security and Safety Issues

Do **not** open public issues for:

- security vulnerabilities  
- unsafe architectural patterns  
- governance violations  

Instead, report privately using GitHub Security Advisories.

---

## 4. Getting Help with Contributions

If you need help contributing:

- read `CONTRIBUTING.md`  
- review the architecture overview  
- ask in GitHub Discussions  

Maintainers will assist when possible.

---

## 5. Response Expectations

This is an open‑source project maintained voluntarily.  
Response times may vary, but maintainers aim to:

- respond to issues within 7 days  
- review PRs within 14 days  
- address security concerns as soon as possible
