# Memory & Knowledge Layer

The Memory & Knowledge Layer manages all forms of memory, knowledge representation, retrieval, consolidation, and safety. It provides the agent with long-term understanding and contextual grounding.

## Responsibilities
- Episodic, semantic, and working memory.
- Knowledge normalization, integration, and grounding.
- Retrieval pipelines and ranking.
- Memory consolidation and summarization.
- Privacy, safety, and access control for stored information.

## Relationships to Other Layers
- **Interaction** — supplies conversational context.
- **Cognitive Planning** — supports reasoning with knowledge.
- **Tooling Execution** — stores results of actions.
- **Safety & Governance** — enforces memory access policies.
